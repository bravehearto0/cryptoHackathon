# cryptoHackathon


## Server

Install truffle
```
npm install -g truffle
```

Go to smart contract folder
```
cd smartContracts
```

Compile
```
truffle compile
```


## Frontend

Install nvm
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash
```

Go into frontend folder
```
cd frontend
```

Install node ^6
```
nvm use
```

Install dependencies
```
npm i
```

Start hacking
```
npm run watch
```

View the app

[open](http://localhost:3000/)

```
open http://localhost:3000/
```
