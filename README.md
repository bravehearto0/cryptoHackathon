## Frontend

Install nvm
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash
```

Install node ^6
```
nvm install 6
nvm use 6
```

Install yarn 
```
npm i -g yarn
```

Go into frontend folder
```
cd frontend
```

Install dependencies
```
yarn
```

Start hacking
```
yarn start
```

View the app

[open](http://localhost:3000/)

```
open http://localhost:3000/
```
